Často kladené otázky

Co je to SplashSMP?
Jsme soukromý Minecraft Roleplay SMP server, který i s pauzami běží od roku 2021. Více se o nás dozvíš zde: Historie
Jak server funguje, role atd.
Server je na styl podobající se TrospySMP (zavádíme větší roleplay). Pluginy, custom svět a některé vylepšené struktury. Snažíme se mít server vyvážený.
Vládne prezident a jeho vláda (zástupce, premiér, soudce apod). Prezident si určí strážce a členy armády. Řídí celý server a snaží se udržet mír, ovšem ne vždy se mu to daří. Je to nejvýše postavený člověk na serveru. Řeší nákup pozemků na spawnu, smlouvy, popř. soudy, eventy a další věci v roleplayi v závislosti na okolnostech. Nevedeme každý den obrovské války, protože by server do měsíce umřel. Máme specifický a hodně velký lore, který poznáš pouze když se napojíš! Právě vykonávají hlavní funkce tito lidé:

Kdy dostanu IP, výběr Discord atd.?
Po úspěšném náboru obdržíš pozvánku na privátní SplashSMP discord, kde najdeš veškeré další informace.
Buguje mi web, nebo nefunguje na mobilu
Ano, náš webhosting je zdarma, proto se může občas bugovat. Zkus CTRL+F5 nebo změnit prohlížeč.
Verze, módy
Server je na Minecraft Java edici 1.21.4. Zakázané módy jsou uvedeny na privátním discordu, kam se dostaneš po úspěšném náboru.
Jakým způsobem mohu podpořit server?
Založ si ticket na public discordu serveru, kde se domluvíme.
